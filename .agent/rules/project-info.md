# Описание проетка

## Краткая информация

Это Nuxt 4 веб-приложение, которое похоже на ChatGPT, но использует вместо GPT модели Gemini (в будущем возможно не только). Этот сайт должен быть очень похож по функционалу с оригинальным ChatGPT (генерация изображений, кода, веб-поиск и и т.д.). Это пет-проект, поэтому он в основном должен работать без базы данных и сервера.

## Tech-stak

Основа — Nuxt 4, Хранение данных — Pinia, Localstorage, Cookie. UI библиотеки — Nuxt UI, TailwindCSS.
